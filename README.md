# E34dash
Simple and dirty tool to edit blue/brown "coding plug" firmware of BMW E32/E34 **HIGH** instrument clusters.

## Allows to edit
* VIN
* Odometer
* Engine type
* Fuel tank type
* Various gauge parameters
* and more!

***NOT*** for **LOW** clusters.
